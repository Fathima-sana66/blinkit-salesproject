# blinkit-salesproject
## Blinkit Sales Performance Dashboard
Project Overview
This project involves a comprehensive analysis of Blinkit's sales data to uncover insights into outlet performance, customer preferences regarding fat content, and regional sales distribution. The dashboard serves as a strategic tool for stakeholders to monitor KPIs and optimize inventory and outlet management.
Key Performance Indicators (KPIs)
Total Sales: $1.20M
Average Sales: ₹141
Number of Items: 8,523
Average Rating: 4.0 / 5
Dashboard Features
1. Sales Distribution & Trends
Outlet Establishment: A line chart tracking sales growth and fluctuations over time (e.g., peak performance reaching $205K).
Total Sales by Item Type: A bar chart identifying top-performing categories like Fruits, Snacks, and Household items.
2. Product & Outlet Segmentation
Fat Content Analysis: A donut chart comparing "Low Fat" (35.4%) vs. "Regular" (64.6%) sales.
Outlet Size & Location: Breakdown of sales across Tier 1, 2, and 3 cities, further segmented by High, Medium, and Small outlet sizes.
3. Detailed Metrics Table
A granular view of different Outlet Types (Supermarket Type 1, 2, 3, and Grocery Stores) evaluating:
Total Sales
Item Visibility
Average Ratings
Technical Stack
Tool: Power BI / Tableau (The interface indicates a Power BI desktop environment).
Data Source: Blinkit transaction and outlet metadata.
Visuals Used: Donut charts, stacked bar charts, line graphs, and KPI cards.
How to Use the Dashboard
Filter Panel: Use the left-hand sidebar to filter by Outlet Location, Outlet Size, or Outlet Type.
Toggle Metrics: Use the buttons above the charts to switch between viewing Total Sales, Avg Sales, or No. of Items.
Drill Down: Click on specific segments (like "Tier 1") to filter the entire dashboard by that selection.
